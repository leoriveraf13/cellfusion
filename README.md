# cellfusion
Cellfusion Web Test

Setup:
The project is divided into two:
*Server: contains all the necessary APIs for the back-end operation
*Front End Project: React project with NextJS

The 'npm install' statement should be run on:
**cellfusion/
**cellfusion/server

Inside the 'sql' folder you can find the queries for the creation of the database and tables related to the project.

Once the project is successfully installed, it should be started from:
++cellfusion: npm run dev
++cellfusion/server: npm start
